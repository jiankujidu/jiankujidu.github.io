步骤三：样本处理通用指南（供您参考与自查）

💡 本步骤仅为指南，无需填写：

以下为样本处理的通用方法。请确保您的样本已按类似方法处理，以保证实验结果可靠性。如果您的样本处理方法特殊，请在步骤一的"备注"栏中说明。

液体样本处理方法（通用）

样本类型关键处理步骤与要点
血清关键：血液自然凝固10-20分钟 → 离心（2-8°C, 2000-3000转/分, 20分钟） → 仔细收集上清。
血浆关键：必须使用抗凝剂（EDTA/肝素钠/柠檬酸钠）混合10-20分钟 → 离心（条件同上） → 收集上清。
尿液、细胞上清等直接按上述条件离心 → 收集上清。

固体/细胞样本处理方法（通用）

样本类型关键处理步骤与样本量要求
组织 步骤：称取1g组织 + 9ml PBS (pH7.2-7.4) → 充分匀浆 → 离心取上清。
⚠️ 样本量要求： 每个检测指标至少需100mg组织，因个体差异，建议提供1g以上。
培养细胞（胞内蛋白） 步骤：PBS重悬细胞(~10^6个/ml) → 反复冻融或超声破碎 → 离心取上清。
🚫 重要禁忌与保存要求：
  • 严禁NaN3（叠氮钠）： 样本中若含有NaN3，会彻底抑制检测所用的辣根过氧化物酶(HRP)活性，导致实验失败。
  • 避免反复冻融： 请将样本分装保存于-20°C或更低温度，确保每个EP管仅冻融1-2次。
  • 以上为通用指南，特殊样本请参考文献处理。
